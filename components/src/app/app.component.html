Behold, this is our component library.

It has a bunch of components in it. This is the app in which we develop and test them, making sure they look and behave the way we expect. It should also be used as a reference guide for developers who want to use them.

Below are the components we have for you, along with some notes about how to use them, or something.

People

Given an array of Person objects, this component renders a list of humans. 

<co-people [people]="people"></co-people>

Person

Given a single Person object, this component renders a single human.

<co-person [person]="person"></co-person>
